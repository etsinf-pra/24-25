FICHEROS DE CÓDIGO

Los ficheros contienen la implementación en C++ de las respuestas del examen de ejemplo. 
En el examen se pide el pseudocódigo, no la implementación.

robin.cpp: códido que corresponde al ejercicio 1 (robin hood hasing para resolver colisiones)

disk.cpp: códido que corresponde al ejercicio 2 (obtener la ocupación total de un disco)

ruta.cpp: código que corresponde al ejercicio 3 (obtención de una ruta -camino-)

caballo.cpp: código que corresponde al ejercicio 4 (caballo que recorre el tablero completo)
