FICHEROS DE CÓDIGO

Los ficheros contienen la implementación en C++ de las respuestas del examen de recuperación. 
En el examen se pide el pseudocódigo, no la implementación.

rayuela.cpp: códido que corresponde al ejercicio 1 (cuckoo hasing para resolver colisiones)

peso.cpp: códido que corresponde al ejercicio 2 (calcular la altura de un árbol)

camino.cpp: código que corresponde al ejercicio 3 (detección de un ciclo en un grafo)

coloreado.cpp: código que corresponde al ejercicio 4 (problema de las ocho reinas)
